# USP 410/510 Urban Data Science

Course website for USP 410/510: Urban Data Science at Portland State University, Spring 2026.

**Live site:** [usp510.github.io](https://usp510.github.io)

## Structure

- `index.html` — Landing page (hand-crafted, not generated)
- `syllabus.md` / `resources.md` — Source content (Markdown)
- `syllabus.html` / `resources.html` — Generated HTML pages
- `week<N>_slides.md` — Lecture slides (Slidev Markdown)
- `week<N>_slides.pdf` — Generated slide PDFs
- `build.py` — Converts Markdown to HTML and exports slide PDFs
- `style.css` — Shared stylesheet
- `package.json` — Node/Slidev dependencies

## Prerequisites

- [uv](https://docs.astral.sh/uv/) (Python dependencies are declared inline in `build.py`)
- [Node.js](https://nodejs.org/) >= 20 and [pnpm](https://pnpm.io/)

## Setup

```sh
pnpm install
```

## Build

```sh
uv run build.py
```

This regenerates:
- `syllabus.html` and `resources.html` from their Markdown sources
- `week<N>_slides.pdf` from each `week<N>_slides.md` via Slidev
- Slide links in the syllabus schedule table (auto-injected where PDFs exist)

## Preview slides locally

```sh
npx slidev week1_slides.md
```

## Editing

- Edit `syllabus.md` or `resources.md`, then run `uv run build.py` to regenerate HTML.
- Edit `index.html` directly — it is not generated by the build script.
- Add new slides as `week<N>_slides.md` at the repo root. The build script will automatically export them to PDF and link them in the syllabus.
