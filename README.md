# USP 410/510 Urban Data Science

Course website for USP 410/510: Urban Data Science at Portland State University, Spring 2026.

**Live site:** [usp510.github.io](https://usp510.github.io)

## Structure

- `index.html` — Landing page / printable flyer (hand-crafted)
- `syllabus.md` — Syllabus source (Markdown)
- `resources.md` — Resources source (Markdown)
- `build.py` — Converts Markdown sources to HTML and generates `flyer.pdf`
- `style.css` — Shared stylesheet
- `syllabus.html` / `resources.html` — Generated from Markdown by `build.py`
- `flyer.pdf` — 1-page printable flyer generated from `index.html`

## Build

Requires [uv](https://docs.astral.sh/uv/). Dependencies are declared inline in `build.py`.

```sh
uv run build.py
```

This regenerates `syllabus.html`, `resources.html`, and `flyer.pdf` from the Markdown sources and `index.html`.

**Note:** `index.html` is hand-crafted and not generated by the build script. Edit it directly.
